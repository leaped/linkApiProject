To start the application, first download all the dependencies with the command "npm i" in the root directory.
After that, it is only necessary to execute the index.js file, also present in the root directory.

The two routes available are as follows:

--To access documents registered in the database / POST
http://localhost:3003/db/findall 

--To access all businesses with "WON" status on pipedrive and register them on bling / POST
http://localhost:3003/refreshWonDeals
